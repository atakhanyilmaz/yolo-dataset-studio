Bu dashboard tarayıcıda çalışır; dosyaları doğrudan diske yazamaz. Aşağıdaki İndir butonlarıyla üç dosyayı indirin ve proje klasörüne kopyalayın.
Klasör kurulumu için Kod Üret → setup_dirs.ps1 indir seçeneğini kullanın, indirilen dosyayı proje klasörüne kopyalayın ve PowerShell'de .\setup_dirs.ps1 çalıştırın.
Gereksinimler
Python 3.10+ CUDA 12.x cuDNN PyTorch (CUDA build)
Adım adım kurulum — Windows PowerShell
1
Klasörleri oluştur setup_dirs.ps1
cd C:\Users\Atakan\OneDrive\Documents\GitHub\iyhss_data_prepare .\setup_dirs.ps1
setup_dirs.ps1 yoksa Kod Üret sekmesinden indirin.
2
PyTorch — CUDA 12.4 build yükle önce bunu yap
pip uninstall torch torchvision torchaudio -y pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124
CUDA 12.7 driver ile cu124 wheel tam uyumludur.
3
Diğer bağımlılıkları yükle
pip install ultralytics pyyaml matplotlib opencv-python
4
CUDA kontrolü yap
python -c "import torch; print('CUDA:', torch.cuda.is_available()); print('GPU:', torch.cuda.get_device_name(0))"
Beklenen çıktı: CUDA: True | GPU: NVIDIA GeForce RTX 4060
5
Dosyaları proje klasörüne koy
C:\Users\Atakan\OneDrive\Documents\GitHub\iyhss_data_prepare\ ├── auto_label.py ← Kod Üret sekmesinden indir ├── config.yaml ← Kod Üret sekmesinden indir └── setup_dirs.ps1 ← Kod Üret sekmesinden indir
Ayrıca best.pt dosyasını şuraya kopyalayın:
C:\Users\Atakan\Documents\iyhss\weights\yolo26s\best.pt
6
Görselleri input klasörüne koy ve çalıştır
cd C:\Users\Atakan\OneDrive\Documents\GitHub\iyhss_data_prepare python auto_label.py
JSON raporu yapıştır
